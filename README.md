# git-pass1year

Show how much code you wrote in the past year

### Prerequisites

**Git** : `brew install git` (for Mac OS)

### Installing

`git clone git@github.com:zkhCreator/git-pass1year.git`

`cd git-pass1year`

`make (install)`

`cd your_git_project`

`git pass1year`
